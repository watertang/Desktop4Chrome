## Chrome Web Store 提交清单

### 必备材料
- [x] 扩展ZIP包（Desktop4Chrome_publish.zip）
- [x] 扩展图标（128x128像素）
- [ ] 至少一张截图（1280x800或640x400像素）
- [x] 简短描述（最多132个字符）
- [x] 详细描述（最多16,000个字符）
- [x] 隐私政策

### 账号准备
- [ ] 创建Chrome开发者账号
- [ ] 支付一次性注册费用（5美元）
- [ ] 完成开发者账号验证

### 扩展信息
- [x] 扩展名称：Ada&Ava Desktop for Chrome
- [x] 版本：1.1
- [x] 类别：选择"生产力"或"个性化"
- [x] 语言：中文（简体）
- [ ] 网站URL（可选）：GitHub仓库地址
- [ ] 支持邮箱：您的联系邮箱

### 权限说明
- [x] storage：用于在本地存储用户设置和偏好
- [x] tabs：用于在用户点击快捷方式时打开新标签页
- [x] host_permissions：用于加载网站图标和背景图片

### 提交前检查
- [ ] 测试扩展在不同设备和浏览器版本上的兼容性
- [ ] 确保所有功能正常工作
- [ ] 检查manifest.json文件是否正确
- [ ] 确保没有控制台错误或警告
- [ ] 验证隐私政策是否符合要求
- [ ] 检查所有描述文本是否有拼写或语法错误

### 提交后跟进
- [ ] 记录提交日期和时间
- [ ] 保存提交确认邮件或ID
- [ ] 准备好回应审核团队可能的问题
- [ ] 设置提醒以跟踪审核进度
