## Chrome Web Store Screenshot Guide

Chrome Web Store requires at least one screenshot, with recommended dimensions of 1280x800 or 640x400 pixels. Below are suggestions for 5 screenshots and setup guidelines:

### Screenshot 1: Main Interface
**Setup Guidelines:**
- Use a high-quality natural landscape background image
- Display time for two cities (e.g., Beijing and New York)
- Set up 3 rows and 6 columns of shortcuts, including common websites (Google, YouTube, Gmail, etc.)
- Ensure the interface is clean, without extra popups or prompts

### Screenshot 2: Edit Mode
**Setup Guidelines:**
- Enter edit mode (double-click anywhere on the page)
- Display the shortcut editing interface
- Showcase the multi-URL setting feature
- Ensure the interface shows visual cues for edit mode (such as grid lines or edit buttons)

### Screenshot 3: City Management
**Setup Guidelines:**
- Open the city management interface
- Display the city search function and search results
- Show the list of added cities
- Ensure the interface displays visual cues for drag-and-drop sorting

### Screenshot 4: Background Settings
**Setup Guidelines:**
- Open the background settings interface
- Show local upload and URL input options
- Display background display method options
- Can show several thumbnail previews

### Screenshot 5: Data Management
**Setup Guidelines:**
- Display the data export/import interface
- Show backup file selection or save dialog
- Ensure the interface is clean and clear, highlighting data management functionality

### Screenshot Creation Tips
1. Use a high-resolution display to ensure screenshots are clear
2. Choose attractive background images to enhance visual appeal
3. Use real website icons and names to increase authenticity
4. Avoid including personal information in screenshots
5. Ensure all text is clearly readable
6. Use a unified visual style for consistency

### Screenshot Size Requirements
- Recommended size: 1280x800 pixels
- Minimum size: 640x400 pixels
- Format: PNG or JPG
- Maximum file size: No more than 1MB per screenshot
